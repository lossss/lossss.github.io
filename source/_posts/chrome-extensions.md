# 默认快捷键
| 快捷键    | 功能       |
|:----------|:-----------|
| alt+left  | 后退       |
| alt+right | 前进       |
| cmd+y     | 历史       |
| cmd+123   | 标签间切换 |

## vimium
>可以像用vim一样使用chrome
### 常用快捷键
| 快捷键    | 功能                           |
|:----------|: -------------------------------|
| b         | 搜索书签                       |
| B         | 搜索书签新标签页打开           |
| F         | 在新标签页打开                 |
| f         | 在当前页打开                   |
| K         | 选中左边一个标签               |
| J         | 选中右边一个标签               |
| r         | reload 当前页面                |
| t         | 新建一个标签                   |
| o         | 搜索在当前页打开               |
| O         | 搜索在新建页打开               |
| yy        | 拷贝当前网址                   |
| x         | 关闭当前页面                   |
| gi        | 定位到第一个搜索框 tab可以切换 |
| command+l | 定位到地址栏(mac自带)          |
