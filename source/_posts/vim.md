vim 中常用的快捷键
<!--more-->
## 移动

| 快捷键 |                      功能                       |
| ------ | ----------------------------------------------- |
| 0      | （数字 0）移动光标至本行开头                    |
| ^      | 移动光标至本行第一个非空字符处                  |
| $      | 移动到行尾                                      |
| gg     | 最上面一行                                      |
| G      | 移动至文件末尾                                  |
| w      | 向前移动一个词 （上一个字母和数字组成的词之后） |
| W      | 向前移动一个词 （以空格分隔的词）               |
| b      | 向后移动一个词 （下一个字母和数字组成的词之前） |

## 浏览文档
| 快捷键 |         功能         |
| ------ | -------------------- |
| (      | 跳转到上一句         |
| )      | 跳转到下一句         |
| {      | 跳转到上一段         |
| }      | 跳转到下一段         |
| [[     | 跳转到上一部分       |
| ]]     | 跳转到下一部分       |
| []     | 跳转到上一部分的末尾 |
| ][     | 跳转到上一部分的开头 |
