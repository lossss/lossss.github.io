##node 安装

使用nvm控制版本
nvm安装 [参考](https://github.com/creationix/nvm)
```bash
#查看当前已安装的版本
nvm ls
# 查看远端版本
nvm ls-remote
# 安装node
# nvm install node
nvm install v8.9.1
# 切换node版本
# nvm use node
nvm use v8.9.1
```

npm
```bash
# npm  升级 
npm install npm@latest
```
国内可以使用 [cnpm](http://npm.taobao.org/)
